## 你如何自己更新网站

如果你只是修改了 `单词测试.html`，需要同步到 GitHub：

1. 打开终端（Terminal）
2. 输入以下命令（一行一行复制粘贴，按回车）：

```bash
cd ~/Documents/coding
cp 单词测试.html index.html
git add -A
git commit -m "更新网站"
git push
```

3. 如果 `git push` 让你输入用户名和密码：
   - 用户名：`Colargmsk`
   - 密码：去 https://github.com/settings/tokens 生成一个新的 token，粘贴进去
   - token 只需要勾选 `repo` 权限

4. 等待 1-2 分钟，刷新 https://colargmsk.github.io/shici/ 就能看到更新了

> **提示**：如果你觉得每次生成 token 太麻烦，可以在终端执行：
> ```bash
> git remote set-url origin git@github.com:colargmsk/shici.git
> ```
> 但这需要先配置 SSH Key（比较进阶，可以让 Claude 帮你设置）。

## GitHub Token 快速生成

1. 打开 https://github.com/settings/tokens
2. 点 **Generate new token (classic)**
3. Note 随便填，Expiration 选 **No expiration**
4. 勾选 ✅ **repo**
5. 点 **Generate token**
6. 复制绿色的 `ghp_...` 字符串
7. 发给我

## 常见问题

**Q: 为什么需要 `index.html`？**
A: GitHub Pages 默认加载 `index.html`。如果只有 `单词测试.html`，用户需要输入完整网址才能访问。复制一份叫 `index.html` 就能自动打开了。

**Q: 更新后网站没变化？**
A: GitHub Pages 部署需要 1-2 分钟。如果超过 5 分钟还没变，检查仓库 Settings → Pages 是否开启。

**Q: 我想让网站用自己的域名？**
A: 可以！买一个域名后，在 GitHub 仓库 Settings → Pages → Custom domain 填写即可。需要帮助可以告诉我。
