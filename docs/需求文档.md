# 需求文档

## 项目目标

"拾词" — 一个帮助高中生检测英语单词掌握情况的网页工具。单 HTML 文件，打开即用。

## 功能清单

### 1. 单词管理
- 输入英文单词和中文释义（多个释义用分号或逗号分隔）
- 单词列表展示，支持搜索、原地编辑、删除
- 复选框多选 + 批量删除
- 一键添加 30 个高中常用示例单词
- 批量导入（粘贴格式化文本）
- 复制 JSON 文本导出（贴到微信即可备份）
- 粘贴 JSON 文本导入（从微信恢复数据）
- 文件下载/上传导入导出

### 2. 单词测试

#### 三种模式
- **📅 每日挑战**：固定题数（默认10题，5-50可调），优先复习到期单词。完成后锁定并显示连胜天数
- **🎯 弱项突破**：只考正确率低于50%的单词，题数可调（默认20题，5-50可调）
- **🆓 自由测试**：不限题数，结束显示结果并计入统计

#### 三种题型（35%/35%/30% 均衡分布）
- 英→中单选
- 中→英单选
- 多词义多选题（金色闪烁标记，醒目提示）

#### 规则
- 干扰选项来自用户单词库
- 答后即时反馈（正确/错误 + 正确答案）
- SM-2 间隔重复算法：答错 3 分钟后重考，答对间隔递增
- 测试中只显示进度条，不显示得分
- 每日挑战/弱项突破中途结束不计入统计
- 自由测试结束计入统计

### 3. 数据统计
- 全局统计卡片：总单词、总正确率、已掌握数
- 掌握度分类：已掌握 / 掌握中 / 需加强 / 未测试
- 单词详情卡片（彩色背景区分掌握度）
- 四种排序方式：正确率 ↑↓ / 测试次数 / 最近测试 / 字母
- 每日打卡连胜记录
- 重置统计数据按钮

### 4. 设置（独立页面）
- 昵称（测试页显示个性化问候语 + 随机名言）
- 每日挑战题数（5-50）
- 弱项突破题数（5-50）
- 每道题选项数量（3-6）
- 外观：浅色 / 深色 / 跟随系统
- 软件作者信息

### 5. 跨设备同步
- 复制 JSON 文本 → 微信文件传输助手 → 粘贴恢复
- 文件下载/上传（电脑端更方便）

### 6. UI/UX
- 纯白/纯黑两种主题
- 响应式设计，手机/电脑均可使用
- 统一 fadeInUp 动画
- 键盘快捷键：数字键选选项，Enter 确认/下一题
- 测试结束确认弹窗
- 无单词时锁定测试并提示

## 非功能需求
- 数据存浏览器 localStorage
- 打开 HTML 文件即可使用
- 可部署到 GitHub Pages
