# 设计规范

## 整体风格

纯白/纯黑极简风格，去除一切不必要装饰。信息层级清晰，留白充足。

## 字体

系统字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`

## 布局

- 最大宽度 640px 居中
- 顶部固定导航栏
- 圆角 10px
- 卡片间距 10px

## 动画

- 统一使用 `fadeInUp`（淡入 + 上浮 8px）
- 时长 0.25s，缓动 ease
- 填充模式 both（防止闪烁）
- 选项按钮级联延迟 0.03s/个
- 按钮按下缩放 0.97x
- 卡片悬停上浮 2px

## 交互

- 按钮悬停变色，点击缩放
- 正确选项绿色高亮
- 错误选项红色高亮
- 多选题金色脉冲动画标记

## 响应式

- 480px 断点
- 手机端输入框纵向排列
- 统计卡片三列网格
